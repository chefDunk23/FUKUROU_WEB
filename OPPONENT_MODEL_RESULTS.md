# 前走メンバーレベル AIサブモデル 検証結果

## モデル概要

- **モデル名**: opponent_model v2
- **目的**: 前走/前々走の相手関係×クラスから、その馬の実力を評価する
- **実装場所**: `pace_bias_ai/opponent_model/`
- **v1（展開×バイアスAI）とは完全に独立した別モデル**
- **LightGBM lambdarank** (objective=lambdarank, group=race_id)
- **学習日**: 2026-06-30

---

## 特徴量 (28列)

### 前走パフォーマンス（最重要シグナル）
- `prev_rank_norm`: 前走着順/出走頭数（正規化）
- `prev_rank`: 前走着順（生値）
- `prev_margin`: 前走勝ち馬との着差（秒）

### 前々走ベースのレースレベル（メイン）
**PIT制約**: 前々走の対戦相手の次走が今走の予測日より前のもののみ

- `prev2_opp_top3_rate`: 前々走の対戦相手の次走3着以内率
- `prev2_opp_top3_count`: 前々走の対戦相手の次走3着以内頭数
- `prev2_opp_count`: PIT基準を満たした対戦相手の頭数
- `prev2_top3_next_avg`: 前々走の1〜3着馬の次走平均着順
- `prev2_top3_next_rate`: 前々走の1〜3着馬の次走3着以内率

**欠損率**: 25.4%（前々走なし24.5% + opponent_nextなし0.9%）
**平均頭数**: 8.7頭（中央値9頭）← 前走ベース比+45%

### 前走ベースのレースレベル（補助）
**PIT制約**: 前走の対戦相手の次走が今走の予測日より前のもののみ

- `prev1_opp_top3_rate`: 前走の対戦相手の次走3着以内率
- `prev1_opp_top3_count`: 前走の対戦相手の次走3着以内頭数
- `prev1_opp_count`: PIT基準を満たした対戦相手の頭数

**欠損率**: 28.1%（前走なし12.6% + opponent_nextなし15.5%）
**平均頭数**: 6.0頭（PIT制約で46.2%が削除される）

### レースレベル × 着差（複合指標）
- `competitiveness_score`: `prev2_opp_top3_rate / (1 + prev_margin)`
  - 前々走のレースレベルが高い（相手が強い）かつ着差が小さい → 高得点
  - 欠損率: 26.1%（prev2とprev_marginの両方が必要）

### クラス情報
- `prev_grade_rank`: 前走クラス序列（G1=1〜条件戦=9）
- `prev2_grade_rank`: 前々走クラス序列
- `grade_change`: 今走vs前走のクラス変化（正=格上げ）
- `cur_class_rank`: 今走クラス序列
- `class_change`, `class_up`, `class_down`, `grade_drop`

### 斤量
- `kinryo_change`: 前走比の斤量変化（kg）
- `kinryo_vs_field`: 今走フィールド内平均との差（kg）

### 条件変化
- `distance_change`, `surface_changed`, `venue_changed`

### 馬属性・レースコンテキスト
- `horse_age`, `dist_cat`, `surface_code`

### 考察

- **前走着順（prev_rank_norm）が依然1位**: 前走パフォーマンスが最も強いシグナル
- **opponent系のSHAP大幅改善**: v1の最高0.021 → v2では0.040（prev2_opp_count、8位）
  - 前走ベースもv1の0.021 → 0.036（+71%）に改善（両者の競合学習効果）
- **competitiveness_score が16位**: レースレベル×着差の複合指標として有効
- **前々走ベース特徴量が7列すべてTOP20入り**: 設計変更の効果を確認

## 欠損率改善の根拠（v1 → v2）

| | 前走ベース(v1) | 前々走ベース(v2) |
|---|---|---|
| opponent_nextなし率 | 15.5% | **0.9%** |
| 合計欠損率 | 28.1% | **25.4%** |
| 平均データ頭数 | 6.0頭 | **8.7頭**（+45%）|

**理由**: 前々走から今走まで1〜2ヶ月以上あるため、対戦相手のほぼ全員が次走を完了している。
前走ベースは最短2週間の間隔もあり、半数近くの対戦相手がまだ次走を走っていない。

## v1との独立性

v1モデル: 展開×バイアスAI（先行力・コーナー位置・トラックバイアス）
v2モデル: 前走/前々走相手関係×クラス（前走着順・着差・対戦相手の次走成績）

**特徴量の重複はゼロ**（horse_ageのみ共通だが学習データ独立）

C期間一致率（v1 vs opponent_model v1実測）:
- 上位5頭の平均重複: 3.19頭（63.9%）
- 上位1頭の一致率: 26.1% → **74%のレースで異なる馬を最上位推薦**
